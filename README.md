
# Hospital Management System
I provides easy access for patients to easily book the available doctors, at suitable time  intervals.
## Tech Stack
- Node.js
- Express
- MySQL
- React (Vite)
- Tailwind CSS

## Features
- Doctor login
- Patient registration
- Appointment booking
- Role-based dashboard

## Setup

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev
